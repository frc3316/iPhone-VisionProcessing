D-Bug iOS Vision
===

## Requirements
- iOS 11+
- Xcode 9+

## Installation
1. Install cocoapods - `[sudo] gem install cocoapods`
2. Install swiftlint - `brew install swiftlint`
3. Install OpenCV w/cocoapods - `pod install`
4. Attach your provisioning profile
5. Run!
